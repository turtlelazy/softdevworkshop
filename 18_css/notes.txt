What is the difference between inline, internal style sheet, and external style sheet designs?
Inline CSS will allow you to directly edit specific HTML elements. Internal CSS will allow you style the entire page by including CSS rules under head element. External CSS is similar to internal but uses the CSS rules linked in a separate .css file.

What is the difference between margin and padding?
Margin is the space around an element's border while padding is the space between element's border and element's content. In other words, padding concerns itself with what's inside and margins concerns itself with what's outside)
